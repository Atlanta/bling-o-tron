Bling-o-tron
============

# Install

```sh
sudo apt-get install build-essential
cd bling-o-tron
npm i
node .
```

# Config reference

Configuration is set in `config.json` :

```json
{
    "prefix": "!",
    "token": "<discord_bot_token>",
    "currency": ":yellow_circle:",
    "googleToken": "<google_api_token>",
    "spreadsheetId": "<spreadsheet_id>"
}  
```
